# Dataset

Este proyecto utiliza un conjunto de 40 resoluciones públicas de amparo directo en materia penal obtenidas del Poder Judicial de la Federación.

Los documentos originales no se incluyen en este repositorio debido a su tamaño y con el objetivo de mantener un repositorio ligero y enfocado en el proceso analítico.

El notebook documenta todo el flujo de extracción, limpieza, transformación y análisis realizado sobre dichas resoluciones.
